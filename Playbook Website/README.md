## Google Forms Integration

To connect the contact form to Google Forms:

1. Create a Google Form with these fields:
   - Name (text)
   - Email (email)
   - Phone (text)
   - Service Interest (dropdown)
   - Message (paragraph text)

2. Get your form ID from the form URL:
   - Form URL: `https://docs.google.com/forms/d/YOUR_FORM_ID/viewform`
   - Replace `YOUR_FORM_ID` in the HTML

3. Get field IDs by inspecting the form HTML:
   - Right-click on each field in your Google Form
   - Select "Inspect Element"
   - Find the `name` attribute (e.g., `entry.1234567890`)
   - Replace the placeholder entries in the HTML
